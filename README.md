# 🤖 CodeAlpha Basic Chatbot

## 📌 Description
This is a simple rule-based chatbot developed in Python as part of the CodeAlpha Python Programming Internship.

## ✨ Features
- Greets the user
- Responds to "How are you?"
- Tells the current time
- Tells today's date
- Introduces itself
- Tells a joke
- Responds to thank you
- Exits when the user types "bye"

## 🛠️ Technologies Used
- Python 3
- datetime module

## ▶️ How to Run

```bash
python chatbot.py
```

## 📷 Sample Conversation

```
You: hello
Bot: Hello! Nice to meet you.

You: what is your name
Bot: My name is CodeAlpha ChatBot.

You: bye
Bot: Goodbye! Have a great day.
```
